Создать систему, в которой множество беспилотных аппаратов действует как единый механизированный разум.

Не стая.  
Не набор устройств.  
А единая воля.

Система локального позиционирования UWB позволяет рою:

- определять положение с высокой точностью;
- сохранять строй даже в условиях отсутствия GPS;
- координировать действия в замкнутых пространствах;
- выполнять задачи разведки, сопровождения и автономного взаимодействия.
